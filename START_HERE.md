## 🚀 Getting Started

### Step 1: Installation (1 minute)

```bash
cd d:\DEV\ms-dashboard
npm install
```

### Step 2: Configuration (1 minute)

```bash
cp .env.local.example .env.local
# Edit NEXT_PUBLIC_API_URL=https://task.moraspirit.com
```

### Step 3: Run (1 minute)

```bash
npm run dev
# Opens http://localhost:3000
```

### Step 4: Test the App (2 minutes)

- ✅ See member list load
- ✅ Click member to select (yellow border)
- ✅ Date defaults to today
- ✅ Pick different date
- ✅ See availability status (green or red)
- ✅ Try error by using wrong API URL

**Total time: 5 minutes!**
